# 왜 어려운가?

## 현실: 단 하나의 경로만 택한다
- 작가가 "이 분기가 최선이다"고 판단 → 거기만 파고드는 것
- 결과: 다른 가능성은 영원히 미지의 영역
- AI로도: ChatGPT는 단 "1개의 이야기"만 만들고 끝

## Dormammu의 차이
- **모든 분기 탐색**: 28개 모든 세계관을 실제로 시뮬레이션
- **자동 품질 검증**: 각 세계의 일관성, 창발성, 설득력을 6차원으로 평가
- **최적화 선택**: 점수 기반으로 가장 흥미로운 결말 자동 선택

> "탐색하지 않으면 What If는 그저 가정일 뿐이다."

# Dormammu = 28개 세계를 동시에 탐색하는 에이전트 팀

> "단 하나의 질문으로, 에이전트 10명이 모든 가능성을 자동으로 평가하고
> 가장 흥미로운 이야기를 찾아냅니다."

## 이게 가능한 이유

1. **프롬프트가 코드** — Python 클래스 대신 Markdown 프롬프트로 에이전트 정의
2. **상태는 파일** — DB 없이 JSON + Markdown 파일만으로 모든 세계 기록
3. **평가가 자동** — 6차원 메트릭으로 각 세계의 품질 객관화
4. **선택이 명확** — 점수 기반으로 최고의 결말 자동 선택

**결과**: 새로운 창작 도메인도 프롬프트만 바꾸면 즉시 적용 가능!

# 한 노드를 처리하는 과정

## 1️⃣ 입력 & 맥락 조립
사용자: "엘빈이 살았다면?" → DFS 오케스트레이터가 현재 노드 맥락 로드

## 2️⃣ 내러티브 생성 (3개 에이전트 병렬)
- 🎭 **Agent Decision** — 캐릭터의 다음 행동 결정
- ⚡ **Interaction Resolver** — 캐릭터 간 상호작용 해결
- 📖 **Narrator** — 이 모든 것을 이야기로 엮기

## 3️⃣ 검증 (2단계)
- 🔒 **Character Validator** — OOC 검증 + 심리적 일관성 패널티
- 📊 **Node Evaluator** — 6차원 메트릭 평가 (Character Fidelity, Foreshadowing, Narrative Flow, Resonance, Emergence, Plausibility)

## 4️⃣ 의사결정
**점수 > 0.3?** → ✅ EXPAND (좋은 분기, 계속 탐색)
**점수 ≤ 0.3?** → ❌ PRUNE (약한 분기, 제거)

> **DFS는 모든 분기를 이 과정으로 자동 처리합니다.**

# 품질을 어떻게 자동화하는가?

## Dormammu의 "Braintrust" — 6차원 메트릭

| 차원 | 가중치 | 설명 |
|------|--------|------|
| **Character Fidelity** | 20% | 캐릭터 성격·동기 재현도 (Big-5 기반) |
| **Foreshadowing** | 20% | 복선 품질과 회수율 |
| **Narrative Flow** | 15% | 조상 노드와의 자연스러운 연결 |
| **Audience Resonance** | 15% | 독자가 흥미로워할 전개 |
| **Emergence** | 15% | 예상치 못한 창발적 사건 |
| **Plausibility** | 15% | 세계관 규칙 내 논리적 타당성 |

**점수 계산**:
- Composite = Σ(각 차원 × 가중치)
- **Score > 0.3 → EXPAND** (좋은 분기, 계속 탐색)
- **Score ≤ 0.3 → PRUNE** (약한 분기, 가지치기)

> "픽사의 Braintrust를 자동화했다."

# 실제 시뮬레이션 결과

## 시나리오
**주제**: "아르민 대신 엘빈 단장을 살렸다면, 104기 훈련병들은 어떤 결말을 맞이하는가?"

**실행 조건**:
- DFS 깊이: Depth 5까지
- 발산점: 시가시나 탈환전 결말부 (850년)
- 원작 준수: 9거인 능력, 부전의 맹세, 유미르 저주 모두 유지
- 에이전트: 10명 병렬 동작

**결과**:
- 🌳 탐색 노드: **28개** (DFS 깊이 5)
- 📝 생성 글자 수: **3.6만자** (모든 노드의 내러티브)
- 📊 평가 글자 수: **6.1만자** (6차원 평가 리포트)
- 📚 생성 + 평가: **9.7만자**
- ⭐ 최적 경로 점수: **0.878⭐** (가중 평가)

# 28개 세계에서 에이전트가 발견한 결말들

## 핵심 캐릭터 5명의 운명

| 캐릭터 | 최종 결말 |
|--------|----------|
| **엘빈 스미스** | 생존 — 초대형 거인으로 각성, 13년의 저주 속에서 인류의 진정한 형태 추구 |
| **에렌 예거** | 억제 — 엘빈의 정보전 통제 아래, 거짓 수용으로 시간벌기 |
| **미카사 아커만** | 분열 — 에렌 보호본능 vs 엘빈 명령 사이에서 내적 갈등, 최종 에렌과 고립 |
| **장 키르슈타인** | 좌절 — "혼자 가지 마"는 호소 거절, 리더십 상실로 무력화 |
| **플로크 포르스터** | 전환 — 엘빈파 → 예거파로의 은폐된 전환, 파벌 분열의 기원 |

## 부수 캐릭터들

- **한지**: 엘빈 중압감 해방 → 진정한 연구자로 복귀
- **리바이**: 자신의 선택(엘빈 선택)이 초래한 결과와 대면
- **사샤/코니**: 엘빈의 체계적 전략으로 생존율 증대
- **히스토리아**: 엘빈의 전략적 판단에 따른 왕가 혈통 도구화

> "에이전트가 발견한 것: 자율적 선택들이 연쇄되며 **자연스럽게** 발생한 파벌 분열"
